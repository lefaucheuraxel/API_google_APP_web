Règles de confidentialité

Cette page décrit comment MonAPP traite les données lorsque tu utilises la connexion Google et les fonctionnalités liées (Calendar, Gmail, People).

Données collectées

Lors de la connexion Google, MonAPP peut accéder à :

  • Ton identité de base (profil, email) via OpenID Connect
  • Ton agenda Google Calendar (lecture et création d’événements)
  • Ton carnet d’adresses (lecture des contacts, si utilisé)
  • Ton Gmail (lecture des labels, envoi d’emails, si utilisé)

Utilisation des données

Les données sont utilisées uniquement pour fournir les fonctionnalités visibles dans l’application : afficher ton profil, interagir avec Calendar, et exécuter les actions Gmail/People demandées.

Stockage des tokens

Après l’authentification, les tokens OAuth peuvent être stockés côté navigateur dans sessionStorage afin de permettre les appels API. Ils ne sont pas stockés en base de données par défaut.

Partage des données

MonAPP ne vend pas et ne loue pas tes données. Les données sont échangées uniquement avec Google pour exécuter les requêtes que tu déclenches.

Sécurité

L’accès est limité par les permissions (scopes) Google. Tu peux révoquer l’accès à tout moment depuis ton compte Google : Sécurité → Accès des applications tierces.

Suppression des données

Tu peux te déconnecter dans l’application, ce qui supprime les informations stockées dans sessionStorage. Pour supprimer l’accès côté Google, révoque l’autorisation de l’application dans ton compte Google.

Contact

Pour toute question liée à la confidentialité, contacte le propriétaire du service.
